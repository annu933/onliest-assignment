# SVG Flower Assignment (React)

This project implements the modifications requested in the SVG flower assessment.

## Changes Implemented

1. Petal color changed from **pink → red**
2. Petal radius increased from **15 → 20**
3. Center color changed from **yellow → orange**
4. Added an extra petal between **top and right petals**
5. Implemented as a reusable **React component**

## Run Project

```bash
npm install
npm run dev
